#FihHuda/فيه هديً

 it is a Muslim application for reading and LIstening quran , reading Azkar , working on adding some of Eslamic Books And other features

## Which tools Used :
* [Material Design]
* [MVVM architecture Pattern]
* [Rebosotiry & singlton design Pattern]
* [Retrofit for dealing with RestApi]
* [Gson converter]
* [Android Services for mediaPlayer handling]
* [RecyclerView & Fragment]
